How it works

Case note

The same finding as prose, for pasting into a case management system. This is the work-product — a decision is only useful if it can be handed to someone else.

Where your data goes

Nowhere. There is no backend, no database and no account. Everything you type is compared in this tab and is gone when you close it.

Reading a document image happens here too — the picture is decoded into a canvas in your browser and never uploaded. The recogniser that reads it is served from this same site rather than a third party, and only downloads when you actually ask to read an image.

The one plain exception: this page is hosted on GitHub Pages, which sees the request for the page itself, as any web host would.

Use synthetic data regardless. This is a demonstration, not a system of record.
