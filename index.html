Thresholds

The name score sets the provisional finding. Field checks can only lower it, never raise it. Both values are recorded in the case note.

Privacy

Nothing you type leaves this tab. There is no backend, no database and no account.

Document images are decoded in your browser and never uploaded. The recogniser is served from this same site, not a third party, and only downloads when you ask to read an image.

The one plain exception: the page itself is hosted on GitHub Pages, which sees the request for it, as any web host would.

Use synthetic data regardless. This is a demonstration, not a system of record.
